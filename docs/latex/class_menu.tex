\doxysubsection{Detailed Description}
Classe \mbox{\hyperlink{class_menu}{Menu}}

A classe Admin\+Menu relaciona-\/se diretamente com a classe \mbox{\hyperlink{class_load_data}{Load\+Data}}, por meio do atributo data. Esta classe configura a UI do programa, toda a lógica do menu e certas operações de sorting/filtragem.
